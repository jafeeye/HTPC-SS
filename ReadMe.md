# HTPCSS (HTPC System Station) v1.6.1
HTPCSS 為家中建置娛樂系統
- 規格 MSI RX460/4G RAM/8G 已經清理好接觸顯示卡與記憶體接點 
- KODI 使用 tvOSX 面板
- 使用 Windows 7 SP1 X64 為底層作業系統
- 使用 RetroBat v3.0
```
安裝路徑
HTPC_SS
  |----- CrazyKTV 
  |----- CustomUI //自訂圖示與背景資料夾
  |----- EmulationStation
    |------ RetroArch
  |----- KODI //HTPC 系統
    |------ Launcher.exe //-p指令KODI為免安裝版本
  |----- KODI.addon //此版本KODI所有相依性套件
  |----- Kodi.Default //此版本乾淨KODI，復原使用
  |----- Tools

```
